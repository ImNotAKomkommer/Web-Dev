# Lab 5 — Online Store (Components & Interaction)

## Description
This project extends the Online Store from Lab 4 by adding:
- Category → Product hierarchy
- Component architecture (AppComponent → ProductListComponent → ProductItemComponent)
- Like functionality
- Delete functionality
- Share buttons (WhatsApp / Telegram)

## Features
- 4 categories (Smartphones, Laptops, Headphones, Tablets)
- 5 products per category (20 total)
- Products loaded from a service
- Parent-child communication using Angular inputs/outputs
- Responsive card layout with CSS Grid
